# Contributing

Thank you for your interest in contributing to Network Privacy Institute.

## Ways to Contribute

- Improve documentation
- Report issues
- Suggest new features
- Submit pull requests
- Help with website content

## Pull Requests

Please:

1. Keep changes focused and well documented.
2. Follow existing project structure.
3. Explain the purpose of your changes.

## Issues

Before opening a new issue, check whether it has already been reported.

We appreciate all contributions, regardless of size.
